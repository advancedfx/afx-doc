# mirv_disable_specmenu

# Syntax

<tt>mirv_disable_specmenu _**0**|1_</tt>

* **Default:** 0

* **0 :** normal operation
* **1 :** disable spectatormenu

Please note: Normally you would prefer using [[mirv_movie_hidepanels|GoldSrc:mirv_movie_hidepanels]] instead.
