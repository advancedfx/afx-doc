# mirv_movie_clearscreen

<tt>mirv_movie_clearscreen _**0**|1_</tt>

* **Default:** 0

Automatic hud removal when filming starts.
(This command sets <tt>hud_draw 0</tt> and <tt>crosshair 0</tt>.)
