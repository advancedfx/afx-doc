# mirv_fx_wh_tint_colorf

<tt>mirv_fx_wh_tint_colorf _fRed_ _fGreen_ _fBlue_</tt>
  
* **Default:** 0.0 0.5 1.0
 
Allows to set a specific color for tinting the walls.
The 3 values are floating point values between 0.0 and 1.0
The order is R G B.
