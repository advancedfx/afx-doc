# mirv_fx_wh_additive

<tt>mirv_fx_wh_additive _0|**1**_</tt>

* **Default:** 1

If enabled this blends the walls additive. The non additive mode (0) might look a bit wrong in some cases due to OpenGL limitations (polygons overlap in wrong order).
