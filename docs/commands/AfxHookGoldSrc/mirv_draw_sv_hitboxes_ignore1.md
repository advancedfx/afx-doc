# mirv_draw_sv_hitboxes_ignore1

<tt>mirv_draw_sv_hitboxes_ignore1 _**0**|1_</tt>

* **Default:** 0

If enabled the player with index 1 (local player usually) will be ignored for hitbox drawing.
