# mirv_fx_noblend

<tt>mirv_fx_noblend _**0**|1_</tt>

* **Default:** 0

* **0 :** disable (blends are not blocked)
* **1 :** enable (blocks screen blends)<br />This will block Counter-Strike 1.6 flash bang blend effects and also fade to black for example.
