# mirv_zoom_digital

<tt>mirv_zoom_digital _[0.0 - **100.0**]_</tt>

* **Default:** 100

Set the amount of of zoom. Values go from 0 to 100.
Digital zoom zooms the game and everything as a whole (HUD, menus, ...).
